# 晨露早报模板 v1.0（定稿版）

## 📋 模板概述

**定位**：实用导向的市场洞察简报  
**目标**：找到市场热点 + 能做出来 + 实用小工具  
**受众**：有产品思维、想快速验证想法的创作者

---

## 🎨 设计规格

### 视觉风格
- **宽度**：700px 固定（邮件客户端兼容）
- **风格**：EDT杂志风（Editorial）
- **主色调**：`#1a1a1a`（深黑）+ `#ff6b35`（活力橙）
- **字体**：Playfair Display（标题）+ Source Sans 3（正文）
- **图片源**：Unsplash（稳定可靠）

### 核心预制件
1. **全宽头图卡片** - 带阴影的圆角卡片
2. **双栏图文** - 左图右文，用于工具介绍
3. **数据展示** - 三栏或双栏数据卡片
4. **功能列表** - 灰色背景的圆角列表框
5. **深色技术块** - 黑色背景，用于GitHub/技术内容
6. **行动清单** - 橙色背景，突出行动项

---

## 📑 内容结构（6大模块）

### 模块1：Header
- 大标题：晨露早报
- 副标题：PRACTICAL GUIDE · 日期
- Slogan：🎯 找市场热点 · 💪 能做出来 · 🛠️ 实用小工具

### 模块2：方向一 · 微信小游戏
**内容要点**：
- 具体游戏类型（如：合成消除类）
- 参考案例（如：羊了个羊、合成大西瓜）
- 大图头图
- 难度等级 + 开发成本
- 5步执行法（详细可落地）
- 晨露分析 + 推荐评级

**图片**：700px宽头图（Unsplash游戏主题）

### 模块3：方向二 · 浏览器插件（强烈推荐）
**内容要点**：
- 小工具类型
- 参考案例（头部插件案例）
- 左图右文布局
- 5个具体可做方向
- 为什么推荐（低门槛高回报）

**图片**：200px宽工具主题图

### 模块4：方向三 · AI提效工具
**内容要点**：
- 工具名称（如：Lovable、v0等）
- 核心功能一句话
- 双栏图文布局
- 实用场景列表
- 适用边界说明

**图片**：300px宽AI/编程主题图

### 模块5：B站实用教程
**内容要点**：
- 2个精选视频
- 每个视频包含：
  - 标题（可点击链接）
  - 作者 + 播放量 + BV号
  - 一句话简介
  - **详细内容总结**（核心！）
  - "可复现"标签
  - "点击观看视频"按钮

**总结格式**：
```
📌 视频核心内容总结：

1️⃣ 项目背景
   • 要点1
   • 要点2

2️⃣ 成本/收入拆解
   • 具体数字

3️⃣ 关键经验
   ✅ 成功经验
   ❌ 踩坑教训

4️⃣ 适合谁看？
```

### 模块6：市场数据 + 行动清单
**市场数据**：
- 双栏数据卡片（如：4亿+月活 / 300%+增长）
- 机会点总结

**行动清单**：
- 橙色背景突出
- 5条具体可执行的事项
- 每条都包含方向+具体动作

## 📝 内容撰写指南

### 选品标准
1. **市场验证过** - 有真实数据/案例支撑
2. **技术可实现** - 明确技术栈和开发周期
3. **成本可承受** - 开发成本<¥3000，时间<1个月
4. **有变现潜力** - 说明变现路径和预期收入

### 分析维度
每个方向都要包含：
- **为什么火**：市场数据/用户痛点
- **难度评估**：星级+预计时间
- **成本明细**：技术成本+美术成本+其他
- **如何开始**：具体步骤（1、2、3...）
- **避坑指南**：常见错误+解决方案
- **晨露点评**：个人分析+推荐度

### B站视频选择标准
- **可复现性**：有源码/详细步骤/能跟着做
- **真实案例**：作者有实际成果（收入数据/用户数据）
- **接地气**：不是高端技术，普通人能做到
- **近期内容**：最好是3-6个月内的
