## 🔴 发现的问题

### 致命 Bug: cleanup.js 条件顺序错误

**位置**: `scripts/cleanup.js` 第 95-110 行

**当前代码**:
```javascript
// P2: 30天删除
if (age > CONFIG.p2_cleanup_days) {  // age > 30
  fs.unlinkSync(filePath);  // 删除
}
// P1: 90天归档
else if (age > CONFIG.p1_archive_days) {  // age > 90
  fs.renameSync(filePath, archiveFile);  // 归档
}
```

**问题**: 100天的文件会进入第一个条件被删除，永远不会执行归档逻辑。

**修复方案**:
```javascript
// 先判断 90天归档
if (age > CONFIG.p1_archive_days) {
  fs.renameSync(filePath, archiveFile);
}
// 再判断 30天删除
else if (age > CONFIG.p2_cleanup_days) {
  fs.unlinkSync(filePath);
}
```
