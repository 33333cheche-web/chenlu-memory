## 1. 产品定位

**一句话描述**：一个融合番茄工作法与轻量任务管理的网页端专注工具，主打「赛博禅意」视觉风格。

**核心体验**：
- 打开页面 → 立刻进入「专注氛围」
- 极简操作，零学习成本
- 视觉治愈，不打扰用户

---

## 2. 目标用户

- 需要提升专注力的办公人群/学生
- 喜欢简约美学、对视觉有要求
- 轻量级使用，不追求复杂功能

---

## 3. 核心功能清单

### 3.1 番茄钟计时（核心）
- **时长选择**：25分钟 / 45分钟 / 60分钟（三选一）
- **计时显示**：大字号居中，倒计时格式 MM:SS
- **进度环**：圆形进度条，随时间流逝
- **操作按钮**：开始 / 暂停 / 重置 / 提前结束

### 3.2 任务清单
- **添加任务**：输入框快速添加，回车确认
- **任务列表**：可勾选完成、可删除
- **专注关联**：可选择当前专注关联哪个任务（非强制）

### 3.3 音效系统
- **背景音乐**：轻柔钢琴 / Lo-fi（开关+音量）
- **白噪音**：雨声 / 森林 / 咖啡厅（三选一+音量）
- **提示音**：专注开始/结束轻柔提示（开关）

### 3.4 数据统计
- **今日统计**：完成番茄数、总专注时长
- **连续天数**：streak 打卡（类似 GitHub 贡献图简化版）

### 3.5 设置
- 默认专注时长
- 音效开关/音量
- 主题切换（深色/浅色，可选）

### 4.3 状态说明

| 状态 | 界面表现 |
|------|---------|
| **准备态** | 显示倒计时、时长选择、开始按钮 |
| **专注中** | 进度环开始走动，显示剩余时间，可暂停/放弃 |
| **完成态** | 轻微庆祝动效，显示本次专注数据，提示休息 |

## 5. 视觉风格参考

### 5.1 核心关键词
**赛博禅意 = 极简 + 自然系 + 呼吸感**

### 5.2 参考来源
- **FlowSpace 原版**（视频已提供）
- **Linear**：暗黑科技感
- **Notion**：极简留白

### 5.3 标志性视觉元素

#### 🌧️ 雨滴玻璃效果（必须）
- **描述**：深色背景上，雨滴从上方自然滑落
- **特征**：
  - 雨滴有透明感，模拟真实水珠
  - 多条雨滴同时下落，速度不一
  - 营造「窗外下雨，室内专注」的氛围
- **参考**：见提供的视频

#### 🎨 配色方向
| 用途 | 参考色值 | 说明 |
|------|---------|------|
| 主背景 | 深色渐变 | #1a1a2e → #16213e 或类似 |
| 强调色 | 青蓝调 | #4ECDC4 / #45B7D1 |
| 文字主色 | 白色/浅灰 | #FFFFFF / #E0E0E0 |
| 文字次色 | 中灰 | #636E72 |

#### ✨ 动画风格
- **呼吸动画**：背景微光缓慢变化（8-12秒周期）
- **雨滴动画**：自然下落，非机械匀速
- **进度环**：流畅过渡，不卡顿

## 6. 交互说明

### 6.1 核心交互流程

```
用户打开页面
    ↓
看到「准备态」界面（雨滴背景 + 倒计时 25:00）
    ↓
选择时长（或保持默认25分钟）
    ↓
点击「开始专注」
    ↓
进入「专注中」状态（进度环走动，背景雨声可选）
    ↓
倒计时结束 → 轻柔提示音 → 显示「完成」
    ↓
记录数据 → 回到「准备态」
```

### 6.2 手势/操作
- **点击**：开始/暂停/重置
- **选择**：时长切换（单选）
- **输入**：添加任务
- **勾选**：完成任务
