## 1. 产品概述

### 1.1 产品定位
一款融合番茄工作法与轻量任务管理的网页端专注工具，主打「赛博禅意」视觉风格，通过轻柔的背景音乐与白噪音营造沉浸式专注氛围。

### 1.2 核心目标
- 提升工作/学习专注力
- 解决任务管理混乱问题
- 培养持续专注的习惯（ streak 机制）

### 1.3 目标用户
需要提升专注力、管理日常任务的办公人群/学生

## 2. 功能需求

### 2.1 核心功能

#### 2.1.1 番茄钟计时
| 功能点 | 说明 |
|--------|------|
| 时长选择 | 三档可选：25分钟 / 45分钟 / 60分钟 |
| 计时控制 | 开始 / 暂停 / 重置 / 提前结束 |
| 倒计时显示 | 大字号居中显示，视觉舒适 |
| 进度环 | 圆形进度条，随时间流逝渐变 |

#### 2.1.2 任务清单
| 功能点 | 说明 |
|--------|------|
| 添加任务 | 快速输入，支持回车添加 |
| 编辑任务 | 点击编辑，失焦自动保存 |
| 删除任务 | 左滑或点击删除图标 |
| 完成任务 | 勾选标记完成，动画反馈 |
| 专注关联 | 可选择关联当前专注任务（非强制） |

#### 2.1.3 音效系统
| 类型 | 内容 | 控制 |
|------|------|------|
| 背景音乐 | 轻柔钢琴 / Lo-fi / 环境音乐 | 开关 + 音量调节 |
| 白噪音 | 雨声 / 森林 / 咖啡厅 | 开关 + 音量调节 |
| 提示音 | 专注开始/结束轻柔音效 | 开关控制 |

**音乐来源方案**: 从爱给网免费专区下载 5-8 首轻音乐，打包到项目 assets，不依赖外部 CDN

### 2.2 数据统计

| 功能 | 说明 |
|------|------|
| 今日统计 | 完成番茄数、总专注时长 |
| 连续天数 | streak 显示（类似 GitHub 贡献图） |
| 历史记录 | 简单列表展示过往专注记录 |

### 2.3 设置项

| 设置项 | 选项 |
|--------|------|
| 默认专注时长 | 25/45/60 分钟 |
| 背景音乐开关 | 开/关 |
| 白噪音开关 | 开/关 |
| 提示音开关 | 开/关 |
| 主题切换 | 深色/浅色（可选） |

## 3. 视觉设计

### 3.1 设计方向
**赛博禅意 = 极简 + 自然系**

参考风格：
- FlowSpace 原版的呼吸感
- 柔和渐变色背景（米白/浅灰/淡蓝渐变）
- 大量留白，呼吸动画
- 圆角设计，柔和阴影

### 3.2 配色方案
| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#F8F9FA` → `#E9ECEF` | 柔和渐变 |
| 强调色 | `#4ECDC4` / `#45B7D1` | 青蓝调， calming |
| 文字主色 | `#2D3436` | 深灰，不刺眼 |
| 文字次色 | `#636E72` | 中灰，辅助信息 |
| 成功色 | `#00B894` | 完成任务 |

### 3.3 核心视觉元素 - 雨滴玻璃效果

**关键特征**（参考视频）：
- 深色背景上，雨滴从屏幕上方自然滑落
- 雨滴有透明感，模拟真实水珠在玻璃上的形态
- 多条雨滴同时下落，速度不一，营造层次感
- 呼吸般的明暗变化，配合专注节奏

**技术实现**：
- CSS 动画 + SVG 雨滴元素
- 或 Canvas 粒子系统
- 多层雨滴：前景（清晰）、中景（半透明）、背景（模糊）

### 3.4 动画效果
- 呼吸动画：背景微光缓慢呼吸（8-12秒周期）
- 进度环：SVG 环形进度，流畅过渡
- 完成任务：轻微庆祝动效（不打扰）

## 4. 技术方案

### 4.1 技术栈
- **前端**: HTML5 + CSS3 + Vanilla JS（轻量，无框架依赖）
- **音频**: Howler.js 或原生 Web Audio API
- **存储**: LocalStorage（本地保存任务和统计）
- **部署**: 静态网页，可部署到 GitHub Pages / Vercel / Cloudflare Pages

### 4.2 项目结构
```
flowspace/
├── index.html          # 主入口
├── css/
│   ├── main.css       # 主样式
│   └── animations.css # 动画
├── js/
│   ├── app.js         # 主逻辑
│   ├── timer.js       # 计时器
│   ├── audio.js       # 音频控制
│   └── storage.js     # 本地存储
├── assets/
│   ├── music/         # 背景音乐文件
│   └── sounds/        # 音效文件
└── README.md
```

### 4.3 数据存储
使用 LocalStorage 存储：
- 任务列表
- 专注历史记录
- 用户设置
- streak 数据

## 6. 交互流程

### 6.1 开始专注
1. 选择时长（默认25分钟）
2. 选择/输入任务（可选）
3. 点击「开始专注」
4. 倒计时开始，背景音乐播放
5. 专注期间可暂停/放弃

### 6.2 专注完成
1. 倒计时归零，播放轻柔提示音
2. 显示「专注完成」动画
3. 自动记录到今日统计
4. 更新 streak（如果今天第一次完成）
5. 提示休息5分钟
