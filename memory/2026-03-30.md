### Noiz API 重大变更
通过对比昨日成功配置和今日测试，发现 Noiz API 已更新：

| 项目 | 旧配置（昨日） | 新配置（今日） | 状态 |
|------|---------------|---------------|------|
| API Endpoint | `/v1/tts` | `/v1/text-to-speech` | ✅ 已更新 |
| API Key 格式 | `fadecd60...` | `ebca...` | ⚠️ 旧 Key 已 401 |
| 请求格式 | JSON | multipart/form-data | ✅ 已适配 |
| Authorization | `Bearer {key}` | `{key}` (无 Bearer) | ✅ 已适配 |

### 根本原因
**Noiz 更新了 API 版本**，导致：
1. 旧 API `/v1/tts` 已废弃，返回 401
2. 新 API `/v1/text-to-speech` 中文克隆效果可能不如旧版
3. 参考音频语言问题 - 如果原音频是英文，克隆中文会有口音
