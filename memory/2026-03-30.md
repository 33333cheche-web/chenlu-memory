### Baby Bot 模型问题根因确认

**最终诊断：**
- `openai/gpt-5.4` API 格式与 OpenClaw 不兼容
- 错误信息：`HTTP 400: Invalid value: 'tool'. Supported values are: 'assistant', 'system', 'developer', and 'user'.`
- 原因：GPT-5.4 不支持 `tool` 角色类型，但 OpenClaw 会发送 tool 调用消息
- 后续 fallback 到 kimi-coding/kimi-for-coding 又超时 630 秒

**需要修复：**
